## Streamlit Web App - Data-driven solution for Brazilian Real Estate Investment Funds (FIIs)

<img src= "images/streamlit.png" 
 width="650" height="400" />

> The project depolyed a **data-driven solution** for analyzing and getting opportunities of Brazilian Real Estate Investment Funds (known as FIIs from Brazilian portuguese - *Fundos de Investimento ImobiliÃ¡rio*). The deployed web-app has two main functionalities: (i) an interactive ranking of FIIs; (ii) an opportunity filter for FIIs that is based on a sector-related strategy. The active prices and indicator is updated on a daily basis. Hence, the **FIIs solution** could be used **to assist** in strategic investment **decisions**.

> FIIs data-driven solution: [pyFII](https://datafii.streamlit.app/)
